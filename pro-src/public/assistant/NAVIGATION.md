# Navigation

| Area | What for |
|------|----------|
| **Today** | Day view, appointments, follow-ups, smart suggestions |
| **Jobs** | Job list and job detail |
| **Customers** | Customer hub, open invoices |
| **Calendar** | Week grid, add/edit appointments |
| **Time** | Clock in/out, timesheets |
| **Chats** | Full assistant history (when enabled in Settings) |
| **Settings** | Company, banks, AI assistant toggle, white-label |

**Job detail:** Job Information section is always visible. Only sections below it collapse.

**Bubble:** Floating chat is a shortcut into the same assistant brain; Chats tab is the full history UI.
