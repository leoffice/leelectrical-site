# Billing

- Estimates and invoices can be built in-app; QuickBooks remains the books backend when synced.
- Payment links use the existing Sola / Cardknox pay path — do not invent new pay URLs.
- **Always confirm with the owner before:** sending an invoice/estimate email, recording a payment, or creating a pay link for a customer.
- Unsent invoice nags skip items older than one year.
